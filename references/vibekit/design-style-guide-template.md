# VibeKit Design Template Use

Official source: `https://raw.githubusercontent.com/MUKE-coder/vibekit/main/design-style-guide.md`

The official template informed the section structure for the project-specific `design-style-guide.md`: philosophy, typography, colour, spacing, layout, components, motion, accessibility and anti-AI-slop rules.

Do not copy the template's example Invoice Pro branding or indigo defaults into this project. The completed project guide is the source of truth.
