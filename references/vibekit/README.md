# VibeKit Reference Summary

Source: `https://github.com/MUKE-coder/vibekit`

VibeKit is used as the project foundation for phase discipline, AI-agent coding rules, registry-first component selection, production review and anti-generic design safeguards.

Important project exceptions:

- Drizzle instead of Prisma
- no payments
- no broad Redis cache
- no public sign-up
- no WhatsApp API or active SMS

Before implementation, the coding agent must refresh the official VibeKit repository and verify the current compatible guidance. The local files in this folder are project-specific summaries, not copied upstream files.
